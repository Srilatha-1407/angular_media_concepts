Media

Media Devices - navigator.mediaDevices provides access to connected media input devices like cameras and microphones.
navigator.mediaDevices.getUserMedia() to request access to a user's camera and/or microphone. This method returns a promise that resolves with a MediaStream object, allowing to capture audio and video.

Media Stream A MediaStream represents a stream of media content, typically audio or video, that's obtained from a device's camera, microphone, or other multimedia sources.
After obtaining a MediaStream using navigator.mediaDevices.getUserMedia(), we can use it to capture video frames or audio data, manipulate the stream, or display it in your application.

Media Recorder MediaRecorder is an API that allows recording of media streams (audio and video) in the browser.
Usage in Angular/TypeScript: With MediaRecorder, we can record the content from a MediaStream object and save it as a file. we can listen for data available events to collect the recorded data, stop the recording, and save it as a file in various formats (like WebM, WAV, etc.).
